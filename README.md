**Teacher-Student-System-REST-API**

_Made with :_

- Node

_Steps to run application :_

> `git clone https://github.com/lokeshchoudhary-lc/teacher-student-mongodb-rest-api.git`

> `npm install`

> `Make you own .env from .evn.example`

> `npm start`

_Npm Packages Used :_

- Expressjs
- mongoose
- https-errors
- cors
- nodemon (dev)
